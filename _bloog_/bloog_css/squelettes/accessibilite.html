Accessibilité et raccourcis clavier

Des raccourcis clavier sont programmés sur la nouvelle version du site :

  • s : Pour aller au contenu principal de la page ;
  • 1 : Retour à la page d’accueil ;
  • 3 : Perdu ? Direction le plan du site ;
  • 4 : Lancer une recherche dans le site ;
  • 6 : Vous amène à cette page d’aide.
Suivez ces instructions pour activer ces raccourcis claviers dans les principaux navigateurs :
  • Internet Explorer sous Windows : Touche Alt et le raccourci clavier, puis la touche entrée.
  • Mozilla, Netscape, K-Meleon ainsi que FireFox sous Windows : Alt et le raccourci clavier.
  • Opera 7 sous Windows et Macintosh, Linux : Maj + Echap et le raccourci clavier.
  • MSIE Macintosh : Ctrl et le raccourci clavier, puis la touche entrée.
  • Safari 1.2 Macintosh : Ctrl et le raccourci clavier.
  • Mozilla, Netscape Macintosh : Ctrl et le raccourci clavier.
  • Galeon Mozilla FireFox Linux : Alt et le raccourci clavier.
