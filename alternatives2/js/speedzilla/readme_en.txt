Installation
~~~~~~~~~~~~

1. Decompress the archive in your site's folder
2. Customize the index.php according to your needs, keeping untouched the code between Speedzilla begin / Speedzilla end

Translation
~~~~~~~~~~~

The script is in English by default.
Yet, if a language file corresponding to the preferred language of the browser, it is automatically selected.

To add a language :
- duplicate the file en.php,
- rename it with two letters corresponding to the language code (e.g. 'it' for Italian)
- translate the text
- place the file in the speedzilla folder

Color customization
~~~~~~~~~~~~~~~~~~~

Use the online configurator http://www.speedzilla.net/ to compose your script colors.
Replace the file custom.css by the generated file once loaded.
