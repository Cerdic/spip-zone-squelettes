Installation
~~~~~~~~~~~~

1. Décomprimer l'archive dans un dossier de votre site nommé
2. Adapter index.php selon votre site en conservant la zone indiquée par Speedzilla begin / Speedzilla end

Traduction
~~~~~~~~~~

La langue par défaut du script est l'anglais.
Cependant si un fichier de langue correspond à la langue préférée présentée par le navigateur, celle-ci est automatiquement sélectionnée.

Pour ajouter une langue :
- copier le fichier fr.php,
- nommer le nouveau fichier avec les deux lettres d'identification de la langue (par exemple 'it' pour l'italien)
- traduire les textes dans la nouvelle langue (attention aux séparateurs comme les apostrophes)
- placer le fichier dans le dossier speedzilla

Personnalisation des couleurs
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Utiliser le configurateur en ligne sur http://www.speedzilla.net/ et composez votre jeu de coloris.
Remplacer le fichier custom.css par le fichier téléchargé.
