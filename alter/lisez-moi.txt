#############################################################
  DOSSIER themes (fourni avec les squelettes ALTERNATIVES) 
#############################################################

Pour documentation complete, voir : 
	http://www.stages.alternatives.ca/article73.html

Ce dossier contient :
- le jeu de squelette 'alternatives' et diverses variantes (themes)

1. POUR ACTIVER LE SQUELETTE ALTERNATIVES (et ses variantes)
------------------------------------------------------------

- Dans l'espace prive, activer le plugin 'sktheme' 
	(Configuration / Gestion des plugins).
- Activer le squelette de votre choix (exemple : 'alternatives').
- Optionnelement, activer un theme de votre choix.

2. POUR PERSONNALISER LES SQUELETTES ALTERNATIVES
-------------------------------------------------

Si vous desirez profiter des mises a jour regulieres du jeu de squelettes
ALTERNATIVES, il est fortement recommande de ne PAS modifier aucun des fichiers 
qui se trouvent dans /themes/alternatives/ ; tout comme il est fortement
conseille de ne pas alterer les fichiers de squelettes de la distribution de
SPIP (qui se trouvent dans le dossier /dist/).

Pour modifier un fichier du jeu de squelettes ALTERNATIVES :

- Dans le dossier /themes/, changer le nom du dossier /perso-template/ sous un autre nom 
  distinctif de vore choix du genre /perso-mon_theme/.
- Dans le fichier theme.xml et plugin.xml de ce nouveau dossier,
  - modifier le <nom> du theme sous le nom de votre choix
  - optionnellment, modifier la <description>
- Pour modifier les styles de base d'ALTERNATIVES
  - creer un dossier /styles/ dans le nouveau dossier
  - creer dans ce dossier, au choix, un fichier
    - habillages.css
    - perso.css
  - incrire dans l'un ou l'autre de ces fichiers les modifications
    ou ajouts de regles de styles desires,  ainsi que les images
    associees dans un dossier /img/.
- Pour modifier n'importe quel autre fichier du jeu de squelette ALTERNATIVES, 
  il suffit de le recopier a la racine de ce nouveau dossier de personnaliation
  et de le modifier selon vos besoins. Il faut seulement respecter la meme
  arboresence de fichiers : exemple /inc/inc-bas.html


3. POUR VISUALIER LES MODIFICATIONS AVANT ACTIVATION PUBLIQUE
-------------------------------------------------------------

Les modifications apportées précédemment n'affectent pas encore le site public.

Vous pouvez, par contre, en prévisualiser les effets en activant 
dans l'espace prive :
  Configuration -> Gestion des themes -> Configuraiton avancee :
    - Activer le switcher de thème : Oui
    - Seulement pour les administrateurs : Oui

Le site public affichera dans le haut a gauche un menu deroulant, disponible
pour les administraterus seulement, vous permettant de selectionner votre 
nouvelle variante du site.

Ainsi, vous pouvez developper des variantes (themes) sans affecter le 
comportement de votre site public pour vos visiteurs.

4. POUR ACTIVER VOS MODIFICATIONS SUR LE SITE PUBLIC
----------------------------------------------------

Dans l'espace prive, allez a :
  Configuration -> Gestion des themes -> Choix theme public :
    - selectionner 'alternatives' comme squellette,
    - selectionner votre nouveau theme (ou habillage).
