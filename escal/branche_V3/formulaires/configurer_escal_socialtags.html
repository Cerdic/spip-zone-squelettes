Si vous avez activé le plugin "Social tags", Escal vous propose 3 choix de sélecteurs que vous pouvez inscrire dans la configuration du plugin Social tags :
- #socialtags-outils pour afficher les icônes dans la barre d'outils d'Escal
- #socialtags-article pour les afficher sous l'icône "Imprimer cet article" dans vos articles.
- #contenu pour les afficher en bas de chaque article
Mais ce ne sont que 3 possibiltés parmi plein d'autres.
