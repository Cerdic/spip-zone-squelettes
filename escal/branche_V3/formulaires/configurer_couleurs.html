#ACTION_FORMULAIRE{#ENV{action}} Vous pouvez changer les couleurs de votre site en remplaçant la valeur par défaut par une autre couleur
- au format hexadécimal -> exemple : #FFFF00
- au format RGB -> exemple : RGB(255,255,0)
- avec une couleur nommée -> exemple : yellow

Je vous conseille vivement d'utiliser le plugin "Palette 1.4.x" qui vous permettra de choisir une couleur dans le sélecteur qui s'affichera dès que vous cliquerez dans une case.

Attention à ce que le texte ressorte bien sur vos fonds et veillez à bien accorder vos couleurs.

Si vous réinitalisez une valeur ou tout, pensez à valider après !

Les textes

Dans la plupart des cas, la couleur des caractères s'adaptera automatiquement à la couleur du fond :
- si le fond est clair, le texte sera en noir
- si le fond est foncé, le texte sera blanc
Il vous reste à choisir vos couleurs pour ...
