#ACTION_FORMULAIRE{#ENV{action}}
Edito : article avec mot-clé "edito". Le titre du bloc sera celui de l'article.
Accès direct : article avec le mot-clé "acces-direct" pour des liens internes. Le titre du bloc sera celui de l'article.
Articles les plus vus : affichage des articles les plus visités.
Identification : pour s'identifier, donc.
Actus : les articles avec le mot-clé "actus" défilent.
Photos au hasard : les images des articles avec le mot-clé "photo-une" défilent.
Sites favoris : les vignettes des sites référencés avec le mot-clé "favori" défilent.
Derniers commentaires : les derniers commentaires d'articles de tout le site.
Evènements à venir : liste les articles des rubriques ou les articles isolés avec le mot-clé "agenda".
Sur le web : le titre et les derniers articles des sites syndiqués.
Statistiques : tout un choix paramétrable plus bas.
Noisette personnelle : articles avec mot-clé "special". Type de noisette paramétrable.

Pour les noisettes suivantes, la configuration se fait dans la partie "Plugins plus"
Calendrier : nécessite le plugin "Calendrier Mini".
Rainette : nécessite le plugin "Rainette". Infos sur la météo d'une ville au choix.
Shoutbox : nécessite le plugin "Shoutbox". Pour discuter en direct.
