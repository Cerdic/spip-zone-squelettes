BliP en quelques mots

BliP est un squelette-plugin sous licence GPL conçu pour fonctionner avec le système de publication français SPIP.

Vous êtes actuellement sur la Boussole de BliP, un ensemble de page conçu pour vous permettre d'accéder rapidement aux l'informations utiles sur le squelette BliP

La documentation est publiée sur cent20.net, et vous pourrez poser vos questions sur le groupe de discussion crée à cet effet.

Le développement lui est réalisé collaborativement sur SPIP-Zone, et pour que vous puissiez vous y retrouvez facilement, une page de raccourcis est disponible.
