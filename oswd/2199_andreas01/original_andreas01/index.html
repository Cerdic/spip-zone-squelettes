andreas01 v2.0
(Jun 25, 2006)

Welcome to andreas01!

This is an open source XHTML/CSS website template, a great tool that makes it possible for people with no web design experience to build high-quality websites. The template is free to download, and you can modify it freely and use it in any way you want to. There are no obligations and no limitations, although I kindly ask you to leave the original design credits in the footer to support my free template design work!

The andreas01 template is a very simple but also very useful standard web layout: a fixed-width 3-column page with a header and a footer. There are thousands of variations of this layout, and this is no perfect solution. But it should be a great starting point for you to customize, play with and add your own personal touch to. And there is a 2-column version included as well if you prefer that.

Open source web design

If you like this layout and would like to use it in any way, you are free to do so. All I ask for is that you leave the "Design by Andreas Viklund" link in the footer of the site, but that is only a request and not any obligation. If you choose to remove the credits and use the template for commercial purposes, I encourage you to consider a donation to support my work: Make a donation through PayPal!

Good luck with your new website!
