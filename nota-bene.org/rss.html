RSS Syndication

Mummy, what's syndication?

Syndication is a way to keep informed of the latest news on a site. You only need to copy a measly link (see below) and paste it into the appropriate reader, and voilà!

Feed readers are a-plenty: for instance if you're using Firefox, you can use the Sarge extension, very handy to see what's new on a site within your browser. You can also create a new «RSS and blogs» account. You can also install a dedicated feed reader. There is a complete page at Wikipedia from which you can learn a lot.

RSS feeds on nota-bene.org

On nota-bene.org you can read the following RSS2 feeds:

  • All the articles
  • Articles in english only
  • The complete blogroll (all the links, in english and in french)
  • Blogroll in english only

Moreover you can also find an RSS feed for every comment thread on the site. You'll find a little 'RSS' link in the 'Comments' block.

Syndication RSS

C'est quoi, dis maman, la syndication ?

La syndication est un moyen de vous tenir informé des dernières nouveautés d'un site. Il vous suffit de copier un petit lien de rien du tout (voir ci-dessous), de le coller dans un lecteur approprié, et le tour est joué.

Les lecteurs de fils syndiqués sont légion : vous pouvez par exemple, si vous utilisez Firefox, utiliser l'extension Sage, bien pratique pour voir directement les nouveautés d'un site sans quitter votre navigateur favori. Vous pouvez aussi dans Thunderbird créer un nouveau compte de type « Nouvelles RSS et blog ». Vous pouvez aussi installer un logiciel spécifique de lecture des RSS. Le mieux est sans doute d'aller lire la page dévolue au RSS sur Wikipedia

Les fils RSS de nota-bene.org

Sur nota-bene.org vous pouvez suivre plusieurs fils RSS2 :

  • Tous les articles du site
  • Uniquement les articles en français
  • Le blogroll complet (tous les liens, en français et en anglais)
  • Uniquement le blogroll en français

De plus vous pouvez aussi récupérer un fil RSS pour les commentaires de chaque article ou brève particulier. Vous trouverez un petit lien 'RSS' du côté des commentaires.
