
Règles d'intervention
=====================

 Pour toute modification dans ce repository svn, vous devez respecter les règles de commit ( de modifications ) suivantes:
 
 - Pour l'instant cet espace n'est pas libre de commit. Seuls les développeurs du squelette Sarka-SPIP peuvent commiter (SarkASmeL, SarkALéoN, SarkARchimeD, SarkAFeeK)
 - Si toutefois vous voulez proposer une évolution ou une correction, adressez-vous d'abord au collectif Sarka-SPIP en nous faisant parvenir un mail à l'adresse eric AT smellup.net.
 - Ces règles sont temporaires, le temps de parvenir à une version aboutie du squelette telle que nous l'avons imaginé. Merci d'avance de les respecter.

Vous pouvez aussi nous retrouver sur le site www.sarka-spip.com ou sur l'IRC, serveur irc.freenode.org canal #spip

Bonne utilisation
